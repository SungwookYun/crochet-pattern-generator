## 🧪 PWA 기능 테스트 방법

### 1. 로컬 테스트:
```bash
cd client
npm start
# http://localhost:3000 에서 테스트
```

### 2. 브라우저 개발자 도구에서:
- **Application** 탭 → **Service Workers** 확인
- **Application** 탭 → **Manifest** 확인  
- **Lighthouse** → **PWA** 점수 확인

### 3. 모바일에서:
- Chrome/Safari에서 "홈 화면에 추가" 버튼 확인
- 설치 후 앱처럼 실행되는지 확인
